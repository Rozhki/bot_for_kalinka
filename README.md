# Tg-бот учёта ИТ-заявок

Бот позволяющий сотрудникам предприятия отправляь заявки по вопросам ИТ-оборудования, программного обеспечения и оргтехники.

# Установка проекта

В терминале VS Code, находясь в папке проекта:

``` 
# Создать виртуальное окружение
python -m venv venv

# Активировать его 
venv\Scripts\Activate.ps1 

pip install -r requirements.txt
```

## Получение токена бота

1. В Telegram найдите бота **@BotFather**.
2. Отправьте команду `/newbot`, укажите имя и username бота (username
   должен заканчиваться на `bot`, например `mzk_it_support_bot`).
3. BotFather выдаст токен вида `123456789:AAH7...`. Сохраните его.

## Получение своего Telegram ID

Самый простой способ - написать боту **@userinfobot**, он пришлёт ваш
числовой ID. Этот ID нужно указать как администратора бота.

## Файл .env

BOT_TOKEN=123456789:AAH7ваштокен
ADMIN_ID=123456789

## Запуск бота

Если всё настроено верно, в консоли появится лог о начале polling'а

При первом запуске автоматически создастся файл `requests.db` со всеми
нужными таблицами - отдельно ничего создавать не нужно.


**Сотрудник:**
- «📝 Создать заявку»
- «📋 Мои заявки» — список своих заявок
- «❓ Помощь» — краткая справка.

**Администратор**
- «📂 Все заявки» / «🆕 Новые заявки»
- «📊 Статистика» — количество заявок по типам проблем.
