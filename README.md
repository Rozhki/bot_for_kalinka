# Tg-бот учёта ИТ-заявок

Бот позволяющий сотрудникам предприятия отправляь заявки по вопросам ИТ-оборудования, программного обеспечения и оргтехники.

# Установка проекта

В терминале VS Code, находясь в папке проекта:

``` 
# Создать виртуальное окружение
python -m venv venv

# Активировать его 
venv\Scripts\Activate.ps1 

pip install -r requirements.txt
```

## Файл .env

BOT_TOKEN=123456789:AAH7ваштокен
ADMIN_ID=123456789

## Запуск бота

Если всё настроено верно, в консоли появится лог о начале polling'а

При первом запуске автоматически создастся файл `requests.db` со всеми
нужными таблицами


**Сотрудник:**
- «📝 Создать заявку»
- «📋 Мои заявки» — список своих заявок
- «❓ Помощь» — краткая справка.

**Администратор**
- «📂 Все заявки» / «🆕 Новые заявки»
- «📊 Статистика» — количество заявок по типам проблем.
